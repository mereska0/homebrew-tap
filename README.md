# Mereska0 Tap

## How do I install these formulae?

`brew install mereska0/tap/<formula>`

Or `brew tap mereska0/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mereska0/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
